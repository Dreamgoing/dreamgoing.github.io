Bloom Filters

简介

Bloom Filter布隆过滤器是一种的数据结构用来判断元素是否在特定的集合之中。常见的场景用于海量数据的去重。Bloom Filter的基础是一个Bit vector

实现

Bloom Filter由一个超大的位数组和若干个互不相关的哈希函数构成。下图表示的是k=3 时的Bloom Filter的确情况， 即有3个hash_func， 通过hash_func_i(x)%m, 得到对应每一位的值，在Bit vector中填入。查找时得到的三位值在Bit vector中为1时，表示Bloom filter中存在对应的key。

def lookup(x):
    found = True
    for hash_func in hash_funcs:
        p = hash_func(x) % m
        found = found && bits[p]
    return found
